# Pantalla inicial de indicaciones

Al abrir `index.html`, la página muestra primero las indicaciones generales de la sesión.

## Iniciar la presentación

Selecciona el botón **INICIO**. Aparecerá una animación breve con el logo de Instituto Cedrus y, al terminar, se mostrará la portada.

También puedes iniciar con cualquiera de estas teclas:

- `Enter`
- `Espacio`
- `Flecha derecha`
- `Flecha abajo`

## Imagen utilizada

La imagen se encuentra en:

```text
assets/indicaciones-generales.png
```

Puedes reemplazarla conservando el mismo nombre de archivo.
